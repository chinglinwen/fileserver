# fileserver

A minimal download fileserver

# Usage

```
Usage of ./fileserver:
  -author
        Show author.
  -path string
        File server path. (default ".")
  -port string
        Port number. (default "9000")
  -v    Show version.
```

# Example usage

```
./fileserver -path public -port 9001 &
```
